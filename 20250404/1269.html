加密软件核心功能demo开发

2025-04-04 14:20:08 

本项目旨在开发一款与现有 Live2D 加密软件类似的加密模型 demo。Demo 程序主要用于验证以下两点：



1，加密后的模型文件能够经过特定流程（加密→虚拟文件系统→文件注入）后，被 VTube Studio（简称 VTS）顺利加载。

2，Demo 程序仅仅用来验证加密文件能够被 VTS 正常读取，与正式产品相比功能较为简单，但需提前预留好未来进阶开发时需要对外暴露的 API 接口。

--

具体需求私信详细沟通，已有类似软件成功实现上述功能。

Windows 系统开发及文件系统原理

有 Dokan/WinFsp、虚拟文件系统或类似项目开发经验

加密算法（例如 AES-256）、安全验证与进程控制经验
