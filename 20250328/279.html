### 项目发布：桌面软件前端开发需求

#### **项目名称**  
重庆某科技有限公司 - 智能设备管理前端系统开发
---
#### **需求描述**
为了提升公司内部智能设备管理效率，我们计划开发一款基于桌面的智能设备管理系统前端。该系统将与后端API对接，用于展示、管理和监控各类智能设备（如智能家居设备、电力设备、车载设备等）的状态和运行数据。
主要功能模块包括：
1. **设备信息管理**：支持设备的添加、删除、修改、查询操作。
2. **实时状态监控**：通过图表或仪表盘展示设备的实时运行状态（如温度、电压、电流等）。
3. **报警提醒**：当设备出现异常时，能够及时弹出报警提示。
4. **历史数据分析**：支持查看设备的历史运行数据，并生成简单的统计图表。
5. **用户权限管理**：根据不同角色分配不同的操作权限。
6. **多语言支持**：支持中文和英文两种语言界面切换。
7. **离线模式**：在无网络情况下，仍可保存部分数据并在网络恢复后同步。
系统需具备良好的用户体验设计，界面简洁美观，响应速度快，支持跨平台运行（Windows/Mac/Linux）。
---
#### **最匹配的技能要求**
1. 熟悉桌面应用开发框架，优先考虑以下技术栈：
   - **Electron** 或 **Qt** 开发经验
   - **React Native Desktop** 或 **Tauri**
2. 掌握前端开发相关技术：
   - HTML5/CSS3/JavaScript
   - TypeScript（加分项）
   - Vue.js 或 React.js（熟悉任一即可）
3. 能够使用数据可视化库：
   - ECharts 或 Chart.js
4. 具备与后端API对接的经验，熟悉 RESTful API 或 GraphQL 的使用。
5. 熟悉多语言国际化方案（i18n）。
6. 对跨平台桌面应用优化有一定了解，确保应用在不同操作系统上的兼容性和性能。
7. 有实际完成过类似桌面应用项目的案例者优先。
