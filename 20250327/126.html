职位概述：




我们正在寻找一位高级移动软件工程师，负责领导我们移动应用的开发和维护，技术栈主要基于 React Native。作为移动工程团队的核心成员，你将负责构建高质量、可扩展且用户友好的应用。


在此岗位中，你将使用 Native Wind 进行样式管理，React Native Reusable 作为 UI 组件库，并依托 Expo 生态系统（Expo EAS、Expo Router、Expo Notifications） 来优化部署、导航和通知功能。此外，你还将通过 Maestro 实现自动化测试，以确保应用的稳定性和可靠性。

1. 移动应用开发：


    

  • 负责使用 React Native 构建、维护移动应用。

  • 运用 Native Wind 实现响应式、高性能的一致性 UI。

  • 实现基于 Expo Router 的高效应用内导航。

  • 集成 Expo Notifications 管理推送与应用内通知。
